# India Village Insights Dashboard 📊

## 📌 Project Overview

This project analyzes village-level data across India using Power BI.

## 🔧 Tools Used

* Power BI
* Excel

## 📊 Dashboard Features

* Total Villages, States, Districts KPI
* Map Visualization
* Top 10 States by Villages
* Top 10 Districts
* Interactive Filters (State & District)
